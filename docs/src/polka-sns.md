# SNS実装としてのpolka

polkaは、分散SNS基盤であり、その基盤をベースとしたSNSとしての実装でもあります。

## polkaのSNS実装

polkaのSNS実装は、中央集計サーバーによるトレンドや数字のない分散型ソーシャルデジタルガーデンです。

## polkaのSNSとしての特徴

### 中央集計サーバー不在

polkaには中央サーバーがありません。各ユーザーが自分のデータを静的ファイルとしてホスティングし、他のユーザーがそれを直接読み取ります。

### トレンドや数値の排除

- いいね数、リツイート数などの集計値なし
- トレンドやアルゴリズム推薦なし

### デジタルガーデン

polkaは「ソーシャルメディア」ではなく「ソーシャルデジタルガーデン」です。

- 投稿は時系列ではなくタグで整理
- 自分のペースで育てるコンテンツ
- フォローはタグ単位(ユーザー全体ではなく、特定の話題のみ)

### 完全なデータ所有権

すべてのデータは単一のCARファイルとして存在し、いつでもエクスポート・移行できます。プラットフォームに依存せず、データを完全に所有できます。
