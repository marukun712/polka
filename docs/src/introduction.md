# polkaとは

**polka**は、タグとリンクでつながる新しい分散型Webプロトコルです。「**検証可能なWeb 1.0**」をコンセプトに、個人サイトの自由さと、分散型技術による信頼性を両立させます。

## 開発の背景：既存Webの課題
現代のWebでは、サービス管理者がデータを保持し、APIを通じて提供しています。しかし、この構造には以下の課題があります。

*   **中央集権的なコントロール**: 運営企業の判断によるアカウントBANや投稿へのアクセス遮断。
*   **プラットフォーム依存**: API制限やサービス終了により、これまでの活動やコミュニティが失われるリスク。

## polkaの3要素
polkaは以下の3つの柱によって、サービスが消えてもデータが残るWebを実現します。

1.  **データとサービスの分離**: ユーザー自身がサーバーでデータを公開し、アプリがそこに直接アクセスする。
2.  **電子署名による検証可能性**: 暗号学的にデータの真正性を保証する。
3.  **タグによるリンクのつながり**: タグを用いたグラフ構造により、コミュニティを自然発生させる。
